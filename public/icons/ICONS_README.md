# PWA Icons

## Current Status
This directory contains SVG placeholder icons. For best results, replace these with actual PNG files.

## Required Files
- `icon-192x192.png` - Used for mobile home screen
- `icon-512x512.png` - Used for splash screen and high-res displays

## How to Create Icons

### Option 1: Use Online Generator
1. Visit https://www.pwabuilder.com/imageGenerator
2. Upload a 512x512 image with your app logo
3. Download the generated icons
4. Replace the SVG files with the PNG files

### Option 2: Design Your Own
Create square PNG images with these specifications:
- 192x192px (72dpi minimum)
- 512x512px (72dpi minimum)
- Transparent or solid background
- Clear, simple icon design
- Safe area: Keep important content in center 80%

### Option 3: Use Icon Tools
- Figma: Export frames as PNG
- Sketch: Export artboards as PNG  
- Photoshop: Save for Web (PNG-24)
- Canva: Download as PNG

## Quick Tip
A Bible book, cross, or "ITI" text logo works great for this app!
